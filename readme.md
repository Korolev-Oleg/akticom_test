### Подготовка
создать в корне файл `.env`
```dotenv
DJANGO_DEBUG=True
DJANGO_SECRET_KEY=<secret_key>
ENABLE_DEBUG_TOOLBAR=False
```

настройка `ENABLE_DEBUG_TOOLBAR=True` замедляет выполнение `bulk_create`
